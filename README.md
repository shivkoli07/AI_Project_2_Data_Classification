#  AI Project - Iris Flower Classification

A Machine Learning classification project developed as part of the **DecodeLabs Artificial Intelligence Internship (Project 2)**.

## 📌 Project Overview

This project classifies Iris flowers into three species using the **K-Nearest Neighbors (KNN)** algorithm.

The model is trained using the famous Iris dataset and can predict the species of a flower based on its measurements.

---

## 🚀 Features

- Load Iris Dataset
- Data Preprocessing
- Feature Scaling
- Train-Test Split
- KNN Classification Model
- Model Evaluation
- Confusion Matrix
- Accuracy & F1 Score
- Predict New Flower Species

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI_Project_2_Data_Classification.git
```

Go into the project folder

```bash
cd AI_Project_2_Data_Classification
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
